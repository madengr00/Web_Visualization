What is the weather like as we approach the equator? This website shows anlalysis of data from the OpenWeatherMap API to model weather relationships to Latitude.

The data analyzed was based on weather in random, normally distributed cities with a population greater than 500, across the globe, using the full range of latitudes and longitudes. Weather outside of a city location was not analyzed, so data leans more toward inhabitable areas.
